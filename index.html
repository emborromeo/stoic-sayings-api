STOIC SAYINGS API

a collection of sayings from ancient stoic philosophers

GET all sayings from ancient philosophers

Description:  This returns all the stoic sayings
Method: GET
URL:  https://stoic-api.herokuapp.com/api/v1/sayings


GET sayings by ID

Description:  This returns a specific stoic saying according to the id
Method: GET
URL:  https://stoic-api.herokuapp.com/api/v1/sayings/:id




GET sayings by philosopher

Description:  This returns all the sayings by each author. We have Aurelius, Seneca &  Epictetus
Method: GET
URL:  https://stoic-api.herokuapp.com/api/v1/:author
